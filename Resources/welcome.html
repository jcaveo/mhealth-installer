mhealth — Mac health & activity dashboard

This installs four background tools that watch system health, log activity, and serve a local dashboard at http://127.0.0.1:8765/.

What gets installed

  • /usr/local/mhealth/ — scripts & plist templates
  • /usr/local/bin/mhealth* — command-line symlinks
  • ~/Library/LaunchAgents/com.mhealth.* — your user's LaunchAgents (3 jobs)

Permissions you'll need to grant

After install, macOS will ask for these the first time each one is needed. You can grant or skip — the tools work either way, but some columns will be blank:

  • Screen Recording → window titles in the activity log
  • Automation → Chrome / Brave → browser tab list
  • Accessibility → frontmost-window detection

To uninstall

Run mhealth-uninstall in Terminal.
