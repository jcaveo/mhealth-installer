mhealth is running.

The dashboard is live at http://127.0.0.1:8765/ — open it in your browser.

Useful commands (Terminal)

  • mhealth — print current health snapshot
  • mhealth-kill — open the dashboard
  • mhealth-setup — re-load LaunchAgents (after updates)
  • mhealth-uninstall — remove this user's setup

Permissions (recap)

Open System Settings → Privacy & Security and grant Screen Recording, Automation, and Accessibility to launchd / Python / bash when prompted. Without them, the tools still run but show fewer details.

Logs

  • ~/Library/Logs/mac-health-watcher.log — every 5 min health sample
  • ~/Library/Logs/mhealth-activity.csv — every 60 s activity row

Both auto-prune to 15 days.
